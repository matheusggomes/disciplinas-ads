TÓPICOS AVANÇADOS EM SISTEMAS DE INFORMAÇÃO I

Ementa

Estudos sobre conceitos e aplicações de Interfaces Gráfcas e utilização de mecanismos de acesso de banco de dados.

Objetivos

  • Conhecer conceitos computacionais no desenvolvimento de programas técnicos científcos
  • Identifcar a codifcação adequada na solução de problemas complexos
  • Conhecer características de linguagens de programação para sistemas computacionais
  • Conhecer algoritmos e técnicas de programação adequadas, para desenvolver aplicativos computacionais
  • Conhecer o paradigma da POO, a atenção a eventos e a programação visual na criação de aplicativos com interfaces gráfcas na web
  • Avaliar necessidades e propor soluções computacionais
  • Aperfeiçoar no uso avançado de uma linguagem de programação
  • Utilizar metodologias adequadas para a solução de problemas
  • Usar computadores, sistemas operacionais, linguagens de programação e ferramentas para o desenvolvimento de aplicativos web
  • Desenvolver aplicativos com interfaces gráfcas utilizando linguagens de programação orientada a objetos
  • Conhecer técnicas de programação para processamento de bancos de dados

Conteúdo

  • Introdução ao desenvolvimento de aplicações web (back-end) com node.js; Arquitetura de micro serviços e arquitetura monolítica.
  • Terminologias por trás do DevOps; Versionamento de código (GitHub); Integração contínua, entrega/implantação contínua (CI/CD); Pipeline CI/CD.
  • Automação; CI/CD (Travis, Jenkins, Gitlab/GitHub, CI/CD etc); Versionamento com GitHub; Descrevendo e documentando API/micro serviços (Swagger, Postman, Kong, etc).
  • Desenvolvimento de aplicações com Node Js; Conceitos de banco de dados relacional e não relacional; Integração do Node JS com MongoDB; Integração do Node JS com MySQL.
  • O que é TDD?; O ciclo do TDD e suas vantagens; Implantando o TDD em projetos back-end usandonode.js; Frameworks de teste (Mocha, Jest, Jasmine, etc).
