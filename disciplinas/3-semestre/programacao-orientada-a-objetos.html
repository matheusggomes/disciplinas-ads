PROGRAMAÇÃO ORIENTADA A OBJETOS

Ementa

Compreensão dos conceitos e aplicações do paradigma da Programação Orientada a Objetos, por meio do desenvolvimento de programas.

Objetivos

  • Conhecer características de linguagens de programação para sistemas computacionais
  • Conhecer o paradigma da programação orientada a objetos
  • Conhecer algoritmos e técnicas de programação adequadas para desenvolver aplicativos computacionais
  • Adquirir conceitos computacionais no desenvolvimento de programas técnicos científcos
  • Conhecer elementos de desenvolvimento de aplicativos com interfaces gráfcas
  • Implementar soluções computacionais de modelagem orientada a objetos
  • Utilizar as características de uma linguagem de programação orientada a objetos
  • Utilizar metodologias adequadas para a solução de problemas com o paradigma orientado a objetos
  • Utilizar ambientes integrados de desenvolvimento (IDE) para edição, compilação, depuração e execução de programas

Conteúdo

  • Aspectos gerais sobre programação de computadores, projeto de sistemas e metodologias; O modelo de objetos.
  • Tipos de dados; Variáveis; Expressões; Classes; Objetos; Atributos; Métodos; Encapsulação de dados; Construtores.
  • Estruturas de decisão simples, composta e aninhada; Tipos de estruturas de repetição; Estruturas de repetição aninhada.
  • Conceitos sobre coleção de dados (vetor e matriz); Atributos de classes como coleção de dados; Coleção de objetos; Recebendo e retornando coleção de dados em métodos.
  • Sobreposição de métodos; Redefnição de métodos; Herança; Relacionamentos de generalização/especifcação; Aplicações utilizando herança.
  • Polimorfsmo; Métodos e objetos polimórfcos; Classes abstratas; Interfaces; Herança por implementação.
  • Instanciação usando Classes Concretas; Instanciação usando Curingas; Classes e métodos genéricos.
  • Referências a objetos; Classes internas; Aplicações utilizado os conceitos de composição e agregação.
  • A Programação Orientada ao Objeto e a utilização de pacotes e classes para o desenvolvimento de interfaces gráfcas; Conceitos gerais de IHC; Componentes de interfaces para criação de rótulos e entrada de texto, painéis ou grupos e botões de comando ou ação; Layouts.
