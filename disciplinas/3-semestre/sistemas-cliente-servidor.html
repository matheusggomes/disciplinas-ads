SISTEMAS CLIENTE/SERVIDOR

Ementa

Estudos sobre programação cliente/servidor, colocando o aluno em contato com novas tecnologias e novos desafos em ambiente de programação.

Objetivos

  • Aprender os paradigmas de desenvolvimento de aplicações para Internet/Web
  • Adquirir o interesse pela pesquisa e pelo conhecimento de novas tecnologias
  • Conhecer processos de elaboração de aplicações Internet/Web
  • Estudar diferentes tecnologias cliente-servidor
  • Reconhecer domínios de aplicação cliente/servidor
  • Saber que tipo de ferramenta de desenvolvimento de software aplicar aos problemas reais
  • Assimilar novos conceitos e ser capaz de acomodá-los acordo com o próprio conhecimento
  • Desenvolver soluções cliente-servidor

Conteúdo

  • Conceitos de Sistemas Cliente/Servidor; Arquitetura de Sistemas: Stand-alone, Sistemas Centralizados, Sistemas Distribuídos e Cliente-Servidor; Modelo de 3 camadas: MVC.
  • Programação em Redes; Sockets; RPC.
  • Introdução; ASP; PHP; NET.
  • Programação de aplicações cliente/servidor na WEB com JSP; Scripts dinâmicos na WEB: Scriptlet, Expressões, Declarações.
  • Banco de Dados em JSP; JavaBeans em JSP; XML em JSP; JavaScript em JSP; Ajax em JSP.
