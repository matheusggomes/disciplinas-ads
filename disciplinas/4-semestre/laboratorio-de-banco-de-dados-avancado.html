LABORATÓRIO DE BANCO DE DADOS AVANÇADO

Ementa

Estudo das linguagens de defnição e manipulação de dados, com ênfase na linguagem SQL. Compreensão de novas tendências na área de banco de dados.

Objetivos

  • Adquirir conhecimentos teóricos e práticos relacionados ao processo de planejamento, construção e manutenção de banco de dados
  • Compreender a arquitetura de um sistema de banco de dados e seu funcionamento
  • Conhecer as tendências acerca de bancos de dados

Conteúdo

  • Revisão SQL Básico
  • Views
  • PL/SQL
  • Stored Procedures
  • Packages – Consultas Complexas
  • Triggers
  • Processamento de Transações
  • Controle de Concorrência
  • Controle de Acesso a Banco de Dados
