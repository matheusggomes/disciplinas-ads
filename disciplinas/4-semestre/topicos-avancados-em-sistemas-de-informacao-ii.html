TÓPICOS AVANÇADOS EM SISTEMAS DE INFORMAÇÃO II

Ementa

Esta disciplina explora os conceitos avançados de desenvolvimento de sistemas de informação utilizando Node.js e seus frameworks, abrangendo arquiteturas de sistemas web, persistência de dados, criação de APIs, otimização para mecanismos de busca, desenvolvimento orientado a testes, controle de versionamento, além de abordar questões de sustentabilidade e legislação pertinentes à área de TI.

Objetivos

  • Compreender os princípios fundamentais de arquiteturas de sistemas web e sua aplicação em projetos utilizando Node.js e frameworks
  • Aplicar os conceitos aprendidos na criação e desenvolvimento de aplicações web utilizando Node.js e seus frameworks associados
  • Analisar as necessidades e requisitos de um projeto de software e selecionar as tecnologias mais adequadas, considerando as melhores práticas e padrões de desenvolvimento
  • Integrar diferentes tecnologias e frameworks para criar soluções complexas e efcientes de software, seguindo padrões de projeto e arquitetura
  • Avaliar a qualidade do código desenvolvido, identifcar possíveis melhorias e aplicar técnicas de otimização para melhorar o desempenho e a escalabilidade das aplicações
  • Dominar as técnicas e práticas avançadas de desenvolvimento de software utilizando Node.js e frameworks associados
  • Capacidade de identifcar e resolver problemas complexos de forma efciente, utilizando ferramentas e técnicas apropriadas
  • Trabalhar em equipe de forma colaborativa, contribuindo com ideias, compartilhando conhecimento e participando ativamente do desenvolvimento de projetos em grupo
  • Demonstrar autonomia na busca por soluções, aprendizado contínuo e desenvolvimento profssional na área de desenvolvimento de software
  • Estar aberto à mudança e adaptar-se rapidamente a novas tecnologias, ferramentas e métodos de trabalho, mantendo-se sempre atualizado no campo da tecnologia da informação

Conteúdo

  • Introdução ao Git e GitHub; Fluxo de trabalho com Git; Colaboração em projetos utilizando controle de versionamento.
  • Introdução aos principais frameworks front-end (ex: React, Angular, Vue.js); Integração de frameworks frontend com Node.js e Express.js; Desenvolvimento fullstack com frameworks back-end e front-end.
  • Model-View-Controller (MVC); Model-View-Template (MVT); Service Oriented Architecture (SOA); Microsserviços; Single Page Applications (SPA).
  • Bancos de dados orientados a documentos (ex: MongoDB); Bancos de dados orientados a objetos (ex: Couchbase).
  • Desenvolvimento de APIs RESTful com Node.js e Express.js; Documentação de APIs; Segurança em APIs na aplicação de Sistema de Informação.
  • Melhores práticas de SEO; Ferramentas de análise de SEO; Otimização de conteúdo para mecanismos de busca.
  • Princípios e práticas do TDD; Ferramentas de teste para Node.js (ex: Mocha, Chai); Integração de testes automatizados no fuxo de desenvolvimento.
  • Testes unitários e de integração; Testes de APIs RESTful; Testes de carga e desempenho.
  • Conceitos de sustentabilidade em TI; Normas e legislação relacionadas à TI Verde; Legislação relacionada à gestão de TI (ex: LGPD, LAI, Lei de Software Livre).
