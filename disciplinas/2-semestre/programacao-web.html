PROGRAMAÇÃO WEB

Ementa

Estudos sobre conceitos, técnicas e tecnologias relacionadas à programação para Internet.

Objetivos

  • Adquirir conceitos básicos sobre a Internet e o ambiente cliente/servidor
  • Estudar as técnicas de desenvolvimento de Scripts (JavaScript)
  • Estudar e utilizar a linguagem JavaScript
  • Conhecer ferramentas de desenvolvimento Web
  • Estudar Documentos Estruturados com XML
  • Utilizar a linguagem JavaScript para processamento no lado cliente
  • Utilizar frameworks de JavaScript
  • Utilizar adequadamente a linguagem XML e/ou JSON e saber aplicá-las

Conteúdo

  • O que é o HTML?; Estrutura e comandos básicos; Formulários em HTML; Introdução ao CSS (Cascade Style Sheet); O que é e para que serve o CSS; Classes, estilos e propriedades em CSS3.
  • Fundamentos da Linguagem; Criação de Scripts; Conceitos de variáveis, operadores, expressões e estruturas de controle; Funções; Tratamento de dados em Formulários; Manipulação de eventos; Array; DOM; Obtendo e alterando atributos; Compatibilidade.
  • JavaScript com HTML5 - Armazenando dados com JSFrameworks (jQuery, modernizr, etc).
  • Introdução a linguagem XML; Utilização de uma linguagem de marcação; Manipulação de XML com JavaScript; Manipulação de JSON com JavaScript; Frameworks atuais utilizados no mercado.
